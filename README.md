## ⚡ Features

### 📱 Device Management
- Auto detect device (ADB/Fastboot/EDL)
- Multi-device support
- Complete device info reading
- USB port detection

### 💾 Backup & Restore
- Full flash (eMMC/UFS) backup
- Partition backup (boot/system/vendor)
- App + data backup
- Contacts/SMS/WiFi backup
- Compressed & encrypted backup

### 🔧 Flashing Engine
- Fastboot flash (single partition)
- EDL 9008 flash (Qualcomm brick recovery)
- Recovery flash (TWRP/OrangeFox)
- MTK SP Flash compatible
- Samsung Odin compatible
- Flash-all script support

### 🔓 Security & Root
- Bootloader unlock/lock wizard
- Magisk boot image patching
- Magisk module management
- LSPosed support
- SELinux toggle

### 🎨 System Customization
- build.prop editor
- DPI/Resolution adjustment
- Boot animation changer
- Font/Emoji replacement
- Hosts ad-blocking

### 🆘 Rescue & Erase
- EDL 9008 rescue mode
- FRP lock removal
- Screen lock removal
- Secure erase (DoD standard)
- Factory reset

### 🖥️ Dual Interface
- **CLI** - Full-featured command line
- **GUI** - Tkinter based graphical interface
